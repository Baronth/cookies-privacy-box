Cookies Privacy Box

- cookies-box.css
- cookies-box.js
- cookies-box.php
- cookies-box-save.php
- cookies-box-config.php

cookies-box.css
:
	Contains the style of the box.

cookies-box.js
:
	Contiene la funzione che crea i cookies e che li salva nel database.
	Viene usato solamente Javascript ( non JQuery ) per poterlo usare anche dove non viene incluso JQuery.

cookies-box.php
:
	Contiene l'html del box. Mostra il box solamente se non è presente il cookie.
	Da cambiare il testo e il link di "maggiori informazioni".

cookies-box-save.php
:
	Salva le informazioni di chi ha cliccato su "OK" in una tabella nel database.
	Salva nel database locale, quindi occorre impostare i campi con i dati del database.
	Crea automaticamente la tabella "cookies-altea" se non è già presente.

cookies-box-config.php
:
	Contiene le informazioni del database.


Funzionamento
:
	Il file cookies-box.php controlla la presenza del cookie "cookieAccepted".
	Se è presente vuol dire che l'utente ha gi� cliccato su "OK" prima.
	Se non è presente mostra il box con un breve testo e il pulsante "OK"
	Cliccando su "OK", la funzione contenuta nel file cookies-altea.js crea il cookie, 
		lo manda all'utente e salva nel database i dati dell'utente.
	I dati vengono salvati attraverso il file cookies-altea-save.php, che deve essere
		modificato inserendo i dati del database del dominio.
	Vengono salvati questi dati: ip, user-agent, pagina in cui � stato cliccato "OK", timestamp.

	Se l'utente non ha attivato Javascript, viene mostrato il box ma non il pulsante "OK".
NB
:
	Il file cookies-altea.js contiene una variabile con il percorso del file cookies-altea-save.php, da
		cambiare per ogni dominio.
	Il file cookies-altea-config.php contiene i dati del database, da cambiare per ogni dominio.
	Il file cookies-altea-config.php deve essere nella stessa cartella di cookies-altea-save.php.
